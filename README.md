# Project 3: Spots

An image sharing site.

##Description

This site is designed for user to upload, discover, and share images. In a safe diverse community to share love through likes and comments. A form of journalism thats tells stories through powerful, honest, and timely photograghs.

###Tech Stack

-HTML
-CSS
-Responsive Designs
-BEM
-Flexbox
-CSS Grid
-Media Queries
